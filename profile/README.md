# dsatechzone-solutions-hub

Engineering organization for **dsatechzone** platform, application, and infrastructure projects.

## Standards

All repositories inherit defaults from our [`.github`](https://github.com/dsatechzone-solutions-hub/.github) repository:

- [Contributing guide](https://github.com/dsatechzone-solutions-hub/.github/blob/main/CONTRIBUTING.md) — TDD, Conventional Commits, code review
- [Release policy](https://github.com/dsatechzone-solutions-hub/.github/blob/main/RELEASE.md) — semantic-release for versioned repos
- [Security policy](https://github.com/dsatechzone-solutions-hub/.github/blob/main/SECURITY.md)
- [Code of Conduct](https://github.com/dsatechzone-solutions-hub/.github/blob/main/CODE_OF_CONDUCT.md)

## Getting started

1. Read [CONTRIBUTING.md](https://github.com/dsatechzone-solutions-hub/.github/blob/main/CONTRIBUTING.md).
2. Open issues using the repository issue templates.
3. Adopt org workflow starters from **Actions → Organization workflows** when creating a new repo.
